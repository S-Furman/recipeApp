# RECIPE APP

Check out the app: https://recipeappbyme.web.app

Simple SPA which allows to store your favourite recipes.

## Tech Stack for this project

- HTML
- CSS
- JavaScript
- React
- React Hooks
- React Router
- Redux
- firebase

### Installation

```

$ git clone https://github.com/S-Furman/recipeApp.git

$ cd recipeApp

$npm install

$npm start
```
